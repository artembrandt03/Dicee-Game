# Dicee (Dice) Game — Mini Web Project
A small and simple **HTML/CSS/JavaScript** project that simulates a two-player dice roll and displays the winner (or a draw).  
This is a lightweight, front-end–only project with no build tools, frameworks, or backend required.
<img width="1180" height="767" alt="image" src="https://github.com/user-attachments/assets/bd645528-fbc9-46aa-bb7f-3677073ef795" />


## Features
- Rolls **two dice** (Player 1 vs Player 2)
- Updates the dice images dynamically
- Displays the result:
  - **Player 1 Wins**
  - **Player 2 Wins**
  - **Draw**
- Clean, simple UI using static assets

## Tech Stack

- **HTML** (page structure)
- **CSS** (styling)
- **JavaScript** (random roll + DOM updates)

## Run Locally
### Option A — Open directly (simplest)
1. Clone the repo
```Bash
git clone https://github.com/artembrandt03/Dicee-Game.git
```
2. Open `dicee.html` in your browser (double-click)

### Option B — Use a local web server (recommended)
Some browsers apply stricter rules when opening files directly. A local server avoids that.

**VS Code Live Server**
1. Install the “Live Server” extension
2. Right-click `dicee.html` → **Open with Live Server**
